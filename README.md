繁草APP是一款生活服务应用，对植物爱好者十分友好。支持植物拍照识别、物种记录管理、分类浏览、画廊浏览、栽种方法查询，以及进行社交分享等功能。

## 1. 拍摄和云端Ai识别
本APP调用android手机的原生相机进行拍摄，并将拍摄图片上传至云端Ai进行识别。
开发默认针对竖屏3:4比例的图片，9:16兼容性正常，其他比例可能需要适当调整。

## 2. 获取植物信息
云端Ai（ doubao-seed-1-6-vision）将返回以下信息：
- 名称
- 描述
- 所属的科、属、种
- 稀有度评定
- 是否有毒物种
- 是否保护物种（中国大陆标准）
- 是否入侵物种（中国大陆标准）

## 3. 获取栽种信息
栽种信息是另一个附加的功能，使用云端Ai（doubao-seed-2-0-mini）进行分析，返回以下内容（简要）：
- 光照
- 温度
- 水分
- 土壤
- 施肥
- 病虫害防治

## 4. 保存、查阅和分享
植物被保存后，可以当作“物种档案”来进行查阅，具体包括分类、画廊两种模式。
此外，单张图片还能进行分享。分享时，原始图片会被包装为一张简约的海报，包括名称、时间、随机的slogan，等。

## 5. 导入导出
APP单独保存图片和植物的相关信息，这意味着图片不会直接存入系统相册。当然你可以手动选择将一些图片另存到系统相册。
通过导入导出功能，植物的数据可以在多个手机之间转移或汇总。

## 6. Ai配置（二次开发）
APP默认统一接入到火山方舟平台，只需要申请一个API-KEY即可。代码中默认使用了doubao-seed-1-6-vision和doubao-seed-2-0-mini，尚未在前端支持更多的大模型。

## 7. 二十四节气海报
仓库里默认附带了76张二十四节气的海报，朋友们可以自行替换为其他海报。
海报的播放规则是：邻近某个节气时，就会循环播放这个节气的海报，一般3-4张图片。
